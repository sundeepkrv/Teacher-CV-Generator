Welcome to Online Resume Builder for Teaching Professionals

This site has been crafted to meet the requirements of a teaching professional to build his/her own online profile which can be hosted on their/college website to have intuitive online presence.

Fill in the below form with your details and click on the Create My Web Profile button on the right side of the page to generate your web profile.

On your generated web profile, click on Web Profile button to download your profile in zip format containing a html file and your uploaded image.

You can edit the html content in your favourite code editor (like VS Code, Sublime Text, Notepad++ or Notepad etc.). Preview your profile in Chrome/Firefox/Safari and adjust for any css styles if not rendered properly.

Use this html code on your website to have an intuitive resume. You can also get the beautiful print of the resume or save as a PDF using Chrome/Firefox.
